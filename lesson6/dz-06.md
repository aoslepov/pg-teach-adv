Создаём 3 вм для кластера etcd  
```
for i in {1..3}; do

  --name etcd-0$i \

  --cores 2 \
  --memory 2G \
  --network-interface subnet-name=default-ru-central1-a,nat-ip-version=ipv4 \
  --zone ru-central1-a \
  --metadata-from-file ssh-keys=/home/aslepov/meta.txt
done
```
